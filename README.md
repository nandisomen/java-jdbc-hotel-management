# Hotel Management System (JDBC)

A console-based hotel reservation management system developed using Core Java, JDBC, and MySQL.

## Features
- Add new reservation
- View all reservations
- Search reservation by guest name
- Update reservation details
- Delete reservation
- Auto timestamp for booking date

## Technologies
- Java
- JDBC
- MySQL

## Key Concepts
- PreparedStatement (SQL Injection Prevention)
- ResultSet Processing
- CRUD Operations
- Exception Handling
- Menu-Driven Console Application

## Database Schema
Table: rev
- rev_id (Primary Key)
- guest_name
- room_num
- contact_num
- rev_date (Auto timestamp)


Future Improvements
- Add room availability check
- Add login authentication
- Convert to GUI or Web-based system
